# Hoofdlettercatalogus gebruiken

De het gebruiksgevallen van de industrie tonen hoe de organisaties in specifieke verticals Adobe Experience Platform en toepassingen toepassen om meetbare bedrijfsresultaten te bereiken. Elk gebruiksgeval beschrijft een concreet bedrijfsscenario, zijn verwacht effect, en verbindingen aan het [ gebruiks gevalpatroon ](/help/blueprints/use-case-patterns/overview.md) dat gedetailleerde implementatierichtlijnen verstrekt.

Blader door de branche om te zoeken naar gebruiksgevallen die relevant zijn voor uw organisatie en volg vervolgens de patroonkoppelingen voor implementatieverwijzingen, zoals richtlijnen voor beslissingen, functieketens en Experience League-documentatie.

| Industrie | Belangrijkste thema&#39;s |
| --- | --- |
| [ Automobielindustrie ](automotive/automotive-overview.md) | Aankoopreis, levenscyclus van de service, ervaringen met verbonden auto&#39;s, loyaliteit van de eigenaar |
| [ B2B ](b2b/b2b-overview.md) | Op rekening-gebaseerde marketing, lood scoring, pijpleidingversnelling, klantenuitbreiding |
| [ Financiële Diensten ](financial-services/financial-services-overview.md) | Aanbevelingen voor producten, kartelpreventie, levensloopaanbiedingen, fraudeverpersoonlijking |
| [ Gezondheidszorg ](healthcare/healthcare-overview.md) | Benoemingsbeheer, therapietrouw, instapweigering van de patiënt, coördinatie van de zorg |
| [ Verzekering ](insurance/insurance-overview.md) | Beleidsvernieuwing, ervaring met claims, risicopreventie, optimalisering van de verschillende verkoopmogelijkheden |
| [ Media &amp; Entertainment ](media-entertainment/media-entertainment-overview.md) | Aanbevelingen voor inhoud, abonnementsbehoud, conversie van proefversies, betrokkenheid voor meerdere platforms |
| [ Detailhandel ](retail/retail-overview.md) | Aanpassing van producten, herstel van winkelwagentjes, optimalisatie van meerdere verkooppunten, betrokkenheid bij loyaliteit |
| [ Telecommunicatie ](telecommunications/telecommunications-overview.md) | Apparaatupgrades, voorkoming van fouten, optimalisatie van plannen, netwerkbetrokkenheid |
| [ Reizen &amp; Ziekenplaats ](travel-hospitality/travel-hospitality-overview.md) | Het boeken van verpersoonlijking, het terugwinnen van het verlaten, loyaliteitsprogramma&#39;s, seizoenscampagnes |

## Hoe gebruiksgevallen zich verbinden met implementatierichtlijnen

Elk gebruiksgeval verbindt met a **gebruiks gevalpatroon** — een herhaalbare implementatiebenadering die de functieketen, de besluitvormingspunten, en de configuratiestappen beschrijft nodig om het gebruiksgeval aan leven te brengen. De gevalpatronen van het gebruik verbinden beurtelings met [ zeer belangrijke bedrijfsdoelstellingen ](/help/blueprints/business-objectives/overview.md), die u helpen implementatiewerk met strategische resultaten richten.

```
Industry Use Case → Use Case Pattern → Key Business Objective
```
